Markdown 语法参考手册

工作中有时候需要记录一些东西，有时候还需要进行有一定格式的书写，本文记录的就是Markdown中的一些语法，学习了以下Markdown语法后，基本可以满足一般的写作

1. 分级标题

使用首行加#号进行书写分级标题

标题H1-H6分别对应为# H1，## H2，### H3，#### H4，##### H5，###### H6

还可以使用 === 表示一级标题，使用 — 表示二级标题。

2. 斜体和粗体

使用 和 * 表示斜体和粗体。

示例：

这是 斜体，这是 粗体。

3. 无序列表(ul)

使用 *，+，- 表示无序列表。

示例：

  • 无序列表项 一
  • 无序列表项 二
  • 无序列表项 三

4. 有序列表（ol）

使用数字和点表示有序列表，数据和.后边需要一个空格

示例：

  1. 有序列表项 一
  2. 有序列表项 二
  3. 有序列表项 三

3. 外链接

  • 文字
    使用 [描述](链接地址) 为文字增加外链接。

示例：

这是去往 本人博客 的链接。

  • 图片
    使用 ![](图片地址)

6. 文字引用

使用 > 表示文字引用。

示例：

野火烧不尽，春风吹又生。

7. 行内代码块

使用 `代码` 表示行内代码块。

示例：

8. 代码块

使用 四个缩进空格 表示代码块。
